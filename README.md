# Facial Recognition using PCA (Eigenfaces)

## Overview

This project implements a facial recognition system using Principal Component Analysis (PCA), also known as the Eigenfaces method. The system reduces high-dimensional image data into a lower-dimensional space for efficient recognition.

## Dataset

AT&T ORL Face Dataset (400 images of 40 individuals)

## Methodology

* Image preprocessing and vectorization
* Mean face computation
* PCA for dimensionality reduction
* Eigenfaces extraction
* Face recognition using Euclidean distance

## Results

* Achieved ~97.5% accuracy using PCA
* Performance varies with number of eigenfaces (k)

## Extension

* PCA + LDA (Fisherfaces) for improved classification

## Tools Used

* Python
* NumPy
* OpenCV
* Matplotlib



# Face-Recognition-PCA
This is my college's first Capstone Project-1. This project(Facial Recognition System) is a system which used to recognize human faces. Facial Recognition System implemented using Principal Components Analysis(PCA) and Eigenfaces method, its reduces high dimensional image data into low dimensional eigenspace and performs face recognition using distance based classification. It is developed on AT & T face dataset and achieves high accuracy in controlled conditions. Implemented from scratch to understand the mathematical foundation of PCA. The project also explores the effect of the number of principal components on performance and extends the approach using Linear Discriminant Analysis (LDA) for improved class separability.
